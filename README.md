# BingoGame

## 概要
Javaで作成したビンゴゲームです。<br>
コンソール上で動作します。<br>

## 機能
### STEP1 ビンゴボールの取り出し
ランダムに数字が出る<br>
一度出た数字は出ない<br>
出た数字を画面に表示する<br>

### STEP2 カードの作成
ランダムにビンゴカードを作成する<br>
カード内に同じ数字は出現せず、以下の数字配置ルールがある<br>
1列目:1～15　2列目:16～30　3列目:31～45　4列目:46～60　5列目:61～75<br>
真ん中はFREE（穴の開いた状態）<br>
ビンゴカードは2次元配列で表現する<br>

### STEP3 ゲームの開始とカードのチェック
開始時にカードと数字を初期化（ランダム作成なので前回と同じにならないこと）<br>
出たボールとカードの状態を表示する<br>
穴の開いた状態でも元の数字が確認できるようにする<br>
一度数字を出すたびに現在のリーチ数、ビンゴ数を表示する<br>
