# BingoGame

## 概要
Javaで作成したビンゴゲームです。
コンソール上で動作します。

## 機能
### STEP1 ビンゴボールの取り出し
ランダムに数字が出る
一度出た数字は出ない
出た数字を画面に表示する

### STEP2 カードの作成
ランダムにビンゴカードを作成する
カード内に同じ数字は出現せず、以下の数字配置ルールがある
1列目:1～15　2列目:16～30　3列目:31～45　4列目:46～60　5列目:61～75
真ん中はFREE（穴の開いた状態）
ビンゴカードは2次元配列で表現する

### STEP3 ゲームの開始とカードのチェック
開始時にカードと数字を初期化（ランダム作成なので前回と同じにならないこと）
出たボールとカードの状態を表示する
穴の開いた状態でも元の数字が確認できるようにする
一度数字を出すたびに現在のリーチ数、ビンゴ数を表示する
